<h2><a href="https://leetcode.com/problems/invert-binary-tree/">
226. Invert Binary Tree
</a></h2>

#### Easy

<hr/>


Given the root of a binary tree, invert the tree, and return its root.


&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; _ 4 _ 
&nbsp; &nbsp; _ 2 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 7 _
_ 1  &nbsp; &nbsp; 3  &nbsp; &nbsp; &nbsp; &nbsp;  6 &nbsp; &nbsp; 9_

<br />
Example 1:
```
Input: root = [4,2,7,1,3,6,9]
Output: [4,7,2,9,6,3,1]
```

Example 2:
```
Input: root = [2,1,3]
Output: [2,3,1]
```

Example 3:
```
Input: root = []
Output: []
```

Constraints:

- ` The number of nodes in the tree is in the range [0, 100]. `
- ` -100 <= Node.val <= 100 `
